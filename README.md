# Техническая составляющая сайта
Написан он на 3 языках <br>
HTML5 - в целом весь сайт написан на нем, семантическая структура страницы + модальные окна и адаптивная верстка. <br>
CSS3 - переключение тем темной/светлой, анимации, кастомные свойства т е переменные.  <br>
JS - Хеширование для админ панели, локальное хранилище, Асинхрованные функции, Drag and Drop для админ панели перетаскивание задач, динамическая генерация, обработка событий, модальные окна. <br>

# По функциональности 
Управление задачами и главами — создание, редактирование, удаление, перенос между главами. <br>
Система валюты (BP) — каждая задача имеет стоимость в BP, которая суммируется в глобальный счётчик. <br>
Бонусные множители — кнопки X2 и VIP увеличивают получаемые BP (вплоть до x4 при совместном включении). <br>
Визуальные статусы — выполненные задачи подсвечиваются, есть анимации появления. <br>
Фильтры — просмотр всех, только активных или только выполненных задач. <br>
Кастомизация видимости — возможность скрывать целые главы или отдельные задачи (настройки сохраняются). <br>
Две цветовые темы — тёмная (по умолчанию) и светлая, переключение с сохранением в localStorage. <br>
Админ-панель с защитой по хешированному паролю — массовое добавление, импорт/экспорт данных, редактирование и удаление. <br>
Drag & Drop — перетаскивание задач между главами для удобной организации. <br>
Адаптивный дизайн — корректно отображается на компьютерах и мобильных устройствах. <br>
