# Техническая составляющая сайта
Написан он на 3 языках
HTML5 - в целом весь сайт написан на нем, семантическая структура страницы + модальные окна и адаптивная верстка
CSS3 - переключение тем темной/светлой, анимации, кастомные свойства т е переменные,  
JS - Хеширование для админ панели, локальное хранилище, Асинхрованные функции, Drag and Drop для админ панели перетаскивание задач, динамическая генерация, обработка событий, модальные окна 

# По функциональности 
Управление задачами и главами — создание, редактирование, удаление, перенос между главами.
Система валюты (BP) — каждая задача имеет стоимость в BP, которая суммируется в глобальный счётчик.
Бонусные множители — кнопки X2 и VIP увеличивают получаемые BP (вплоть до x4 при совместном включении).
Визуальные статусы — выполненные задачи подсвечиваются, есть анимации появления.
Фильтры — просмотр всех, только активных или только выполненных задач.
Кастомизация видимости — возможность скрывать целые главы или отдельные задачи (настройки сохраняются).
Две цветовые темы — тёмная (по умолчанию) и светлая, переключение с сохранением в localStorage.
Админ-панель с защитой по хешированному паролю — массовое добавление, импорт/экспорт данных, редактирование и удаление.
Drag & Drop — перетаскивание задач между главами для удобной организации.
Адаптивный дизайн — корректно отображается на компьютерах и мобильных устройствах.
